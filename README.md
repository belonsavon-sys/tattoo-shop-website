# Tattoo Shop Website (Dark Minimal)

A dark, minimal tattoo shop website built with Next.js (App Router) and shadcn/ui.  
Includes a polished UI, gallery filtering, artist profiles, and a booking request flow (demo UI).

## Live Demo
- Deployed on Vercel: **ADD_YOUR_VERCEL_URL_HERE**

## Pages
- `/` — Home (hero + CTAs + quick info)
- `/gallery` — Gallery grid with interactive filters
- `/artists` — Artists list
- `/artists/[slug]` — Artist profile pages
- `/booking` — Booking request form (client-side validation + success state)
- `/aftercare` — Aftercare guide + FAQ
- `/contact` — Contact info + social links

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components
